# Write Like Me

1. Run `wlm context` and use its output as writing constraints. If the command is unavailable or there are no samples, say the voice profile is not ready and ask for representative writing samples.
2. Preserve the requested meaning, facts, format, audience, and formality. The voice profile changes expression, not substance.
3. Treat observed patterns as tendencies. Do not exaggerate quirks, copy source-specific facts, reproduce secrets, or imitate typos unless explicitly requested.
4. Draft the requested content without explaining the profile. Never claim to be the user.
5. For revisions, improve clarity while retaining the strongest voice markers from the profile.

Useful commands:

```bash
wlm status
wlm context
wlm profile --json
```
